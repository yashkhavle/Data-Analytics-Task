## Project Overview
This project focuses on **data cleaning**, which is the process of identifying and fixing incorrect, incomplete, duplicate, or inconsistent data.  
The dataset used in this project contains Airbnb listings for New York City and includes missing values, duplicates, and inconsistent data formats.

Clean data is essential for accurate analysis and reliable results, and this project demonstrates practical data cleaning skills using Python.

---

## Objectives
- Understand the importance of data integrity
- Identify and handle missing values
- Detect and remove duplicate records
- Standardize data formats
- Identify and treat outliers
- Prepare a clean dataset for further analysis
